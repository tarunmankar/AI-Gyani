# AI-Gyani 🧠

**Apni Bhasha Mein AI Seekho** — A premium tutorial blog platform dedicated to making Artificial Intelligence accessible to everyone in Hinglish.

## 🚀 Quick Start
1. Install dependencies: `npm install`
2. Run locally: `npm run dev`
3. Build for production: `npm run build`
